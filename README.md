# Figma Style Design Tool (DOM Based)

A mini Figma-like design editor built using HTML, CSS, and Vanilla JavaScript.

## Features
- Rectangle & Text creation
- Layers panel
- Properties panel
- Export to JSON & HTML

## Tech Stack
- HTML
- CSS
- JavaScript (DOM only)

## How to Run
Just open index.html in browser.
